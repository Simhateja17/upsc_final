# Roadmap

## Active Milestone: App Backend Integration

### Phase 01: App Backend Integration
**Status:** Planning
**Goal:** Connect the Jeet mobile app to the unified Express backend, enabling real data across all app screens.

**Requirements:**
- [APP-01] Ktor HTTP client with auth interceptors
- [APP-02] Supabase Auth integration (login/signup/token refresh)
- [APP-03] Dashboard, user profile, settings APIs
- [APP-04] Daily MCQ, daily answer, editorial APIs
- [APP-05] Mock tests, PYQ, flashcards, mindmap APIs
- [APP-06] Videos, library, current affairs APIs
- [APP-07] Study planner, syllabus tracker APIs
- [APP-08] AI chat, analytics, billing APIs
- [APP-09] Error handling, loading states, empty states
- [APP-10] End-to-end testing of critical flows
